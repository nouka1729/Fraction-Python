# Fraction Python

Pythonで動く分数ライブラリです

## ドキュメント

### `frac(a,b=1)`

$\displaystyle\frac{a}{b}$ として初期化します

### `四則演算 ・ 比較`

```
+ , -, * , / , += , -= ,*= ,/=

< <= > >= == !=
``````

を使うことができます



### `a.floor() , a.ceil() `

$a$ の繰り下げ、繰り上げを int 型で返します

### `a.bunsi , a.bunbo `

$a$ の分子、分母を int 型で返します

### `a.to_string()`

aの分子、分母をそれぞれn,mとしたときに "n/m" という文字列を返します
